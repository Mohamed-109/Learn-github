1- git clone with link to have the repo locally on your device.

2- git status to know the status of the repo in the file if we did not add any file

3- git add to add fiels to the repo in the statge part

4- git reset head to remove a file from statge part

5- git commit -m "write the message that you want" to move from statge area to local repo

6- git branch to know the branch (master) 

7- git push to push data to remote repo (origin main)

8- git pull to get data from remote repo (origin) to local repo
